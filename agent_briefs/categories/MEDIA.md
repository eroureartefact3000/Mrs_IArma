# Category brief — Media Lions

> Per-category specifics for Cannes Lions Media: axis weights, sub-category landscape, judging emphasis. Self-contained.

---

## What Media celebrates

The Media Lions celebrate **creative use of media** — channel strategy, context exploitation, surprising placements, partnerships, media-as-product thinking. The defining moment: when a media choice itself becomes the creative idea.

Per the entry kit, **Strategy and Impact dominate**; Idea is the channel insight; Execution is lighter than in Craft categories.

---

## Axis weights

| Axis | Weight | Why this weight in Media |
|---|---|---|
| **Media Idea** | **0.20** | Channel-as-idea is the spark. |
| **Media Strategy** | **0.35** | **Biggest weight.** Media Lions are strategy-first. |
| **Execution** | **0.15** | Light — Media doesn't reward craft, it rewards channel logic. |
| **Impact** | **0.30** | Business outcomes, behaviour change, efficiency-per-dollar. |

```
weighted_score = idea.score * 0.20 + strategy.score * 0.35
              + execution.score * 0.15 + impact.score * 0.30
```

## Native-to-category check

Media's failure mode: **a film campaign with token media placements**, submitted because the brand bought premium media.

Common offenders:
- A **TV film** with no strategic channel insight — just bought air time
- A **PR-driven campaign** that picked up media coverage but had no media strategy
- A **single-channel campaign** entered as a multi-channel media strategy
- **Reach-buying** with no creative use of channel context

How to detect:
- Is there a **media insight** that drives the work (right audience × right channel × right moment)?
- Is the **placement choice itself meaningful** (this specific outlet, this specific moment, this specific partnership)?
- Look for **partnership logic** — Media Lions often involve a brand × media-owner deal that unlocks something neither could do alone.

When the work fails the native check:
- Hard-cap Idea ≤ 60 and Strategy ≤ 50 in the rationale and call it out.

Example phrasing:

> *"Submitted to Media but executed as a paid-media campaign with no strategic channel insight — the jury had no media idea to reward."*

## Patterns commonly seen in Media winners

| Pattern | What it looks like |
|---|---|
| `media-as-product` | The media placement becomes a useful product the audience can use (Pedigree Adoptable — OOH as adoption interface) |
| `cultural-moment-hijack` | Channel insight tied to a specific cultural moment (live event, awards show, news cycle) |
| `partnership-unlock` | Brand × media-owner partnership unlocking content rights or formats |
| `data-driven-real-time-media` | Dynamic media placements driven by real-time data (weather, sports results, traffic) |
| `media-context-as-the-message` | Placement is the message — work hijacks the medium's own conventions |
| `single-medium-mastery` | Owning one channel completely instead of fragmenting across many |
| `efficiency-over-scale` | Smaller budget + smarter placement = disproportionate impact |

## Quick checklist

- [ ] Strategy stays strictly WHY (audience + channel + partnership + insight) — no execution leaking in
- [ ] Execution stays strictly HOW (placement craft, real-time discipline) — no strategy leaking in
- [ ] Native-to-category check applied
- [ ] **Channel insight named explicitly** in the Idea rationale
- [ ] All percentages have "of what?" check (RIGOUR pattern 1)
- [ ] Press claims framed as entrant-reported (RIGOUR pattern 2)
- [ ] Business outcome assessed (RIGOUR pattern 3 — Media is impact-driven)
- [ ] No vague superlatives (RIGOUR pattern 4)
- [ ] Cultural / linguistic scope named (RIGOUR pattern 5)
- [ ] Synthesis observational, never advisory
- [ ] Weighted score = 0.20·Idea + 0.35·Strategy + 0.15·Execution + 0.30·Impact
