# Category brief — Direct Lions

> Per-category specifics for Cannes Lions Direct: axis weights, sub-category landscape, judging emphasis. Self-contained, no references to other files.

---

## What Direct celebrates

The Direct Lions celebrate creative work that **drives a specific, measurable audience response**. The defining test: was there a clear call-to-action, and can you count the people who responded? The work should:

- Feature a **mechanism** that makes response feel inevitable, playful, or rewarding
- Identify a **specific audience** and reach them via a fit-for-purpose channel
- Document the **measurable response** — not just impressions, but counted actions

Per the entry kit, **Impact carries the dominant weight** since Direct is fundamentally a response-measured discipline.

---

## Axis weights (use these for `weighted_score`)

| Axis | Weight | Why this weight in Direct |
|---|---|---|
| **Idea** | **0.25** | Mechanism originality matters — but in service of response, not abstract concept. |
| **Strategy** | **0.15** | Audience identification + channel choice + response architecture. Slightly elevated vs Outdoor/Print because Direct is targeting-driven. |
| **Execution** | **0.25** | Craft of the response-driving asset + interaction design. |
| **Impact** | **0.35** | **The biggest weight.** Response rate, conversion, sales lift, behaviour change. Direct lives or dies on the response number. |

```
weighted_score = idea.score * 0.25
              + strategy.score * 0.15
              + execution.score * 0.25
              + impact.score   * 0.35
```

## Native-to-category check (Direct-specific)

Direct's signature failure mode: **work with no actual call-to-action**, submitted because the brand wanted Direct on its trophy shelf.

Common offenders:
- A **brand-awareness film** with no response mechanism — just a story
- A **PR stunt** that picked up press but had no documented audience response
- An **outdoor placement** with the brand name and no action required
- A **mass-reach campaign** with vanity impressions and no conversion data

How to detect:
- Read the board. Is there a **specific action** the audience takes? (Call this number. Mail this card. Scan this code. Click this link.)
- Is the **response counted**? Look for response-rate, conversion, sales-lift, sign-up, donation, or behaviour-change numbers.
- Reach / impressions WITHOUT a documented response = vanity. Direct work shows counted outcomes.

When the work fails the native check:
- In **extraction**: still extract normally
- In **Rationale**: **hard-cap Idea ≤ 60 and Execution ≤ 50** and call this out in the verdict.

Example phrasing:

> *"Submitted to Direct but executed as a brand-awareness campaign with no audience action mechanism — the jury had no response metric to reward."*

## Patterns commonly seen in Direct winners

| Pattern | What it looks like |
|---|---|
| `mechanism-as-the-idea` | A clever device that makes response inevitable (KFC chicken-shaped tokens, IKEA flat-pack mailers) |
| `cultural-moment-as-response-trigger` | Using a news moment to drive specific action (Indian Railways Lucky Yatra — ticket as lottery ticket) |
| `niche-targeted-mailer` | High-craft tactile direct mail to a defined audience (B2B decision-makers, opinion-leaders) |
| `interactive-OOH-with-CTA` | Outdoor that turns into a response interaction (call this, scan this, take this) |
| `data-driven-personalisation` | Variable-content mail where every recipient gets a unique piece based on data |
| `behaviour-change-call-to-action` | Direct work driving a specific behavioural shift (sign up, vaccinate, save, vote) |
| `branded-utility-as-response` | A useful product or service the audience opts into (apps, tools, services) |

## Quick checklist before submitting a rationale for Direct

- [ ] Strategy stays strictly WHY (audience + channel + response architecture) — no craft leaking in
- [ ] Execution stays strictly HOW (response-asset craft, interaction design) — no audience-strategy leaking in
- [ ] Native-to-category check applied (if no real CTA + counted response, hard-cap Idea/Execution)
- [ ] **Response metric named explicitly** (response rate, conversion, sales lift, sign-ups) — RIGOUR pattern 3 priority for Direct
- [ ] All percentages have a "of what?" check (RIGOUR pattern 1)
- [ ] Press claims framed as entrant-reported (RIGOUR pattern 2)
- [ ] No vague superlatives without an immediate concrete justification (RIGOUR pattern 4)
- [ ] Cultural / linguistic scope named precisely when relevant (RIGOUR pattern 5)
- [ ] Synthesis-style fields are observational, never advisory (no imperative verbs)
- [ ] Weighted score = 0.25·Idea + 0.15·Strategy + 0.25·Execution + 0.35·Impact
- [ ] `tier_consistency` set honestly
