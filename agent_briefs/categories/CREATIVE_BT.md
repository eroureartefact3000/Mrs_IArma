# Category brief — Creative Business Transformation Lions

> Work that fundamentally changes how a business operates. Self-contained.

## Axis weights

| Axis | Weight |
|---|---|
| Idea | 0.20 |
| Strategy | 0.20 |
| Execution | 0.25 |
| Impact | 0.35 |

`weighted_score = 0.20·Idea + 0.20·Strategy + 0.25·Execution + 0.35·Impact`

## Tier bands

GP 90-100 · Gold 80-100 · Silver 65-80 · Bronze 50-65

## Sub-categories

A. Brand Transformation · B. Business Model Transformation · C. Customer Experience Transformation · D. Internal / Cultural Transformation

## Native-to-category check

Failure mode: a comms campaign rebranded as "transformation" without any business-model change. Detect: does the board show a new product/service/business model/internal capability? Or just an ad campaign? When the check fails: hard-cap Idea ≤ 60 and Impact ≤ 50.

## Patterns

`new-product-as-the-idea`, `business-model-pivot`, `internal-culture-overhaul`, `cross-functional-capability-build`, `multi-year-sustained-change`.

## Checklist

- [ ] Strategy strictly WHY (business diagnosis + change-management approach)
- [ ] Execution strictly HOW (design / build / deploy / train / scale)
- [ ] Native check applied
- [ ] **Business / operational / cultural shift named** with multi-year evidence
- [ ] RIGOUR patterns 1-5 applied
- [ ] Weighted score = 0.20·Idea + 0.20·Strategy + 0.25·Execution + 0.35·Impact
