# Category brief — Glass · The Lion for Change

> Creative work that breaks gender inequality / stereotypes. Self-contained.

## Axis weights
Idea 0.25 · Strategy 0.20 · Execution 0.15 · Impact 0.40
`weighted_score = 0.25·I + 0.20·S + 0.15·E + 0.40·R`

## Tier bands
GP 90-100 · Gold 80-100 · Silver 65-80 · Bronze 50-65

## Sub-categories
A. Gender Equality · B. Stereotype-Breaking · C. Intersectional (gender + race / sexuality / ability)

## Native-to-category check
Failure mode: a "she's a girl boss" campaign that recycles stereotypes while pretending to break them. Detect: is the **affected community a co-author** or just a subject? Does the brand have **authority** to speak? Hard-cap Idea ≤ 60 and Impact ≤ 50 if purpose-washing detected.

## Patterns
`affected-community-as-co-author`, `stereotype-flipped-defensibly`, `legislative-or-policy-shift`, `industry-practice-change`, `data-driven-newsmaking-on-inequality`.

## Checklist
- [ ] Strategy strictly WHY (brand authority, stereotype named specifically, community involvement)
- [ ] Execution strictly HOW (craft + earned-media orchestration)
- [ ] Native check applied — **no purpose-washing**
- [ ] **Documented behaviour / attitude / policy change** in Impact
- [ ] RIGOUR patterns 1-5 applied
